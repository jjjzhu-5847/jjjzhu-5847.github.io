collection: portfolio --- ### Unity Mini Action-Platformer Prototype #### Project Overview Developed a mini action-platformer prototype using Unity, featuring custom-designed characters, environment assets, and fully animated sprite sheets. #### Character Design & Animation - **Custom Characters:** Designed and animated two original characters using hand-drawn pixel art: - **Main Character (MC)** - **Enemy Character (EC)** - **Animations:** Each character has unique animations for idle, movement, and attack states. #### Environment & Assets - **2-Point Perspective Crate:** Created a dynamic crate asset to enhance gameplay. - **Stylized Background:** Developed a visually cohesive background environment that complements the game's aesthetic. - **Asset Integration:** Imported all assets into Unity and configured sprite animations. #### Gameplay Implementation - Implemented basic gameplay functionality that enables dynamic interactions between characters. - The result is a fully playable demo scene showcasing the complete integration of art, animation, and gameplay mechanics. ---
